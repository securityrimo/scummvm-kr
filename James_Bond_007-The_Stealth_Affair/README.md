## 필수 파일 목록

한글화 적용을 위해 **게임 폴더**에 아래 4개 파일이 필요합니다.

| 파일명 | 설명 |
|--------|------|
| `korean.ini` | 환경 설정 파일 — 색상 · 폰트 · 크기 · 박스 조정 |
| `Koreans.trs` | 대사/나레이션/스토리 번역 파일 (`원문 \t 번역`) |
| `KoreanItems.trs` | 아이템 이름 번역 파일 (`원문 \t 번역`) |
| `Galmuri9.ttf` | 전체 한글 출력용 TTF 폰트 |

> **전용 ScummVM 빌드 필수:** `scummvm_cine_kor_0.8.3.exe`  
> 한글화 파일 미적용 시 → 영어 원본으로 동작합니다.
